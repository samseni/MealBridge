## рџ”§ Backend Updates

### New API Endpoints Added
**File:** `server/src/controllers/admin.controller.js`

1. **`getVerificationRequests()`** (lines 387-418)
   - Get all verification requests with status filter
   - Returns user details, org name, documents, admin notes
   - Supports filtering by pending/approved/rejected/all

2. **`reviewVerification()`** (lines 420-464)
   - Unified endpoint for approve/reject with admin notes
   - Mandatory admin note validation
   - Updates user verification status
   - Saves review timestamp and notes

3. **`getAnalytics()`** (lines 466-563)
   - Platform analytics with date range filtering
   - Returns comprehensive statistics:
     - User counts by role
     - Listing and claim statistics
     - Top donors and NGOs with ratings
     - Waste prevented calculations
   - Default: last 30 days

4. **`exportReport()`** (lines 565-644)
   - Export platform data as CSV or PDF
   - Includes listings and claims within date range
   - CSV format: proper headers and data formatting
   - PDF format: simple text-based report
   - Returns file download
